# DZ-intro-VM
An introductory Virtual Machine activity designed to guide new Linux users through a multitude of command-line interface (CLI) commands and techniques.


## VM Download Link
Download the provided virtual machine for this challenge [here]([url](https://drive.google.com/file/d/1kBrtRAYVXIAtLf6tezk0p12jvpLqcksK/view?usp=sharing)). 


## Challenge Information
This challenge is designed as a beginner-friendly Capture-the-Flag (CTF). All flags will follow the format FLAGX{Flag_String_Here}, where X represents the flag number,
corresponding to the respective challenge number. There are 10 challenges and 10 flags total. 

**EVERYTHING** for this challenge can be completed through the terminal, with the tools provided. There is no need to install new tools or utilize the GUI for any reason.

There are **two separate handouts** for this challenge. You'll notice they're nearly identical, with a slight distinction: The Beginner version (denoted by '(B)' at the end of the filename) is designed for complete beginners, and will "hold your hand" through the challenges, so to speak. The veteran version (V) is designed for users with some experience, and will provide more vague hints that leave the challenger to figure out which commands they need to use for any given challenge.
